# Система для проведения статистических экспериментов

Система предназначена для автоматизации проведения статистических экспериментов и анализа гипотез. Пользователь определяет конфигурацию эксперимента, включая тип распределения, применяемые статистические критерии и параметры выборки. На основе этой конфигурации система выполняет полный цикл: генерирует случайные данные в соответствии с выбранным распределением, применяет статистические тесты, вычисляет критические значения и сохраняет результаты.

Архитектура системы построена многоуровневой. На уровне конфигурации определяются все параметры эксперимента. Слой генерирования отвечает за создание случайных выборок нужного размера. Вычислительный слой реализует статистические расчёты и применение критериев согласия. Все результаты сохраняются в системе хранилищ, которая обеспечивает персистентность и организованное управление данными различных типов.

Выполнение структурировано через последовательность шагов: генерирование данных, вычисление критических значений, анализ мощности и оценка временной сложности. Система поддерживает параллельное выполнение задач через рабочих процессов, что обеспечивает эффективную обработку больших объёмов вычислений. По завершении эксперимента система генерирует детальные отчёты, содержащие ключевые метрики и результаты анализа.

Основная цель — обеспечить возможность масштабируемого проведения статистических исследований без необходимости ручной реализации каждого вычисления, позволяя сосредоточиться на анализе и интерпретации результатов.
