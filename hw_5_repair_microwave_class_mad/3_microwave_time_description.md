# Временная диаграмма сценария разогрева еды

## Описание сценария
Диаграмма показывает временную последовательность работы микроволновки в сценарии типичного разогрева еды.

## Три параллельные шкалы времени

**Door (Дверца):**
- Начально: Opened (открыта)
- В момент closeDoor: переходит в Closed (закрыта)
- Остаётся Closed во время приготовления (cooking timer)
- В момент openDoor: переходит обратно в Opened (открыта)

**Magnetron (Магнетрон):**
- Начально: Off (выключен)
- В момент startCooking: включается (On)
- Остаётся On во время приготовления
- В момент finishCooking: выключается (Off)

**Light (Свет):**
- Начально: On (включен)
- В момент closeDoor: выключается (Off)
- Включается On во время приготовления
- В момент finishCooking: выключается (Off)
- В момент openDoor: включается обратно (On)

## Ключевые моменты времени

1. **closeDoor** - пользователь закрывает дверцу (дверца переходит из Opened в Closed)
2. **startCooking** - пользователь запускает приготовление (магнетрон включается, свет включается)
3. **cooking timer** - период активного приготовления (магнетрон работает, свет включен)
4. **finishCooking** - время приготовления истекает (магнетрон выключается, свет выключается)
5. **openDoor** - пользователь открывает дверцу (дверца переходит из Closed в Opened, свет включается)

## Сценарий в хронологическом порядке

1. Дверца открыта, свет включен, магнетрон выключен (начальное состояние)
2. Пользователь закрывает дверцу -> свет выключается
3. Пользователь запускает приготовление -> магнетрон включается, свет включается
4. Происходит процесс приготовления в течение cooking timer -> магнетрон остаётся включённым
5. Таймер истекает (finishCooking) -> магнетрон выключается, свеь выключается
6. Пользователь открывает дверцу -> свет включается, еда готова

## Логика работы

Диаграмма демонстрирует корреляцию между состояниями трёх компонентов: дверца управляет светом (свет горит только дверь открыта и когда еда разогревается), а магнетрон работает только во время приготовления с установленным таймером.
