# Тестирование

## Запуск тестов

```bash
# Все тесты
make test

# Конкретный пакет
go test ./internal/builtins/...

# С покрытием
go test -cover ./...
```

## Структура тестов

Тесты организованы по пакетам и покрывают основные компоненты:

- **builtins** - встроенные команды (cat, echo, wc, pwd, exit)
- **lexer** - токенизация входных строк
- **parser** - парсинг токенов в AST
- **executor** - выполнение команд и пайплайнов
- **environment** - управление переменными окружения
- **shell** - основная оболочка и обработка команд

## Линтер
